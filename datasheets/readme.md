# ThomsonLint datasheets folder

Place local component datasheets here before running ThomsonLint.

This folder is for user-supplied reference datasheets that support BOM and component evidence review. ThomsonLint should check this folder before attempting any web-based datasheet discovery.

## What to place here

Use this folder for PDFs or other locally saved reference files for specific BOM parts, such as:

- IC datasheets
- regulator datasheets
- connector datasheets
- sensor datasheets
- optocoupler / isolation component datasheets
- MOSFET / diode / transistor datasheets
- module datasheets
- manufacturer application notes when relevant to the design review

## Recommended filename format

Use clear filenames that include the manufacturer and part number when possible:

```text
TI_LM2596_datasheet.pdf
ST_STM32F407VG_datasheet.pdf
Molex_1050170001_datasheet.pdf
Wurth_74437324022_datasheet.pdf
